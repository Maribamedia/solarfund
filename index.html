To qualify for IPP funding consideration, a project must meet the following minimum requirements. Incomplete submissions will not be reviewed.

Executive Summary - Project Introduction
Site Location & Title Deed overview
Structure of the deal / PPA finance
Pre-Construction & Implementation status: (EIA, Finalise Technical Design, Municipal Approval, Property Lease, Final BQO, PPA Detail, CEL, Budget Quote, Final Project Plan, Commissioning stages / Project Phases e.g. 5MW at a time)
Proposed Consortium Team
Organogram showing different key parties involved
Project Schedule (detailed timeline)

Scoping report
Water Use Licence verification
Any application for sub-divisions of land
Grid connection agreement / feasibility (where applicable)
Financial model (CAPEX, OPEX, IRR, debt service cover)
Off-taker credit assessment or sovereign guarantee outline
